# Smart Energy Monitoring and Predictive Maintenance System

An end-to-end industrial IoT and machine learning architecture for monitoring electrical parameters and predicting equipment failure.

## Architecture
1. **Hardware:** ESP32 for high-frequency electrical sensor data acquisition.
2. **Edge Gateway:** Containerized MQTT broker for data routing.
3. **ML Engine:** Python-based deep learning anomaly detection.
4. **SCADA:** Dashboard for real-time visualization and alarm management.
